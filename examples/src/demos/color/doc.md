## Color 色彩

### 主题色

:::default
<color-primary></color-primary>
:::

### 辅助色

除了主题色外的场景色，需要在不同的场景中使用

:::default
<color-assist></color-assist>
:::

### 背景色

卡片或其他容器的背景色

:::default
<color-background></color-background>
:::

### 字体色

字体的基本用色

:::default
<color-font></color-font>
:::

### 边框色

边框的基本用色

:::default
<color-border></color-border>
:::

### 投影

投影的基本用色

:::default
<color-shadow></color-shadow>
:::
